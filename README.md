## 🏠Projet

## Projet 是什么

Projet 是一个`在线生成 H5 页面并提供页面管理和页面编辑的平台`，用于快速制作 H5 页面区块。用户无需掌握复杂的编程技术，通过简单配置即可制作精美的页面，可用于营销场景下的页面区块制作。同时，也为开发者提供了`编程接入能力`

## 支持功能

- [x] 基础组件，复合组件，组件嵌套
- [x] 实时预览
- [x] 样式编辑
- [ ] 组件保存
- [ ] 模版保存
